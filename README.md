# Track_file:

### Содержит класс Track с 4 атрибутами:

  - name
  - duration
  - genre
  - rating

# Playlist

### Содержит класс перечесления RepeatMode:

  NONE - повтор треков выключен
  ONE - повторится текущий трек
  ALL - повторятся все треки
  
### Содержит класс Playlist с 4 атрибутами:

  - name
  - _tracks   (массив содержащий треки)
  - _current_index (индекс текущего трека в массиве)
  - repeat_mode

  И 7 фукнциями:
  - add_track 
  - delete_track
  - shuffle_playlist
  - current_track - получение текущего трека, возвращает экземпляр Track
  - next_track - получаем следующий трек (он тановится текущим)
  - display - вывод плейлиста ввиде таблицы, задание 3
  - find_tracks_in_time_range - поиск в плейлисте n треков с длительность между указанными значениями, возвращает список с именами треков, задание 6
